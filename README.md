# Browser-Extension-Develop-Scaffold

## Doc references
- [Chrome_developer_Document](https://developer.chrome.com/docs/extensions/reference/)

## Project dependencies
- node v18.x
- webpack v5.x
- typescript v5.x

## NPM 

1.Getting started
```shell
npm install
```
2.Develop a extension
```shell
npm run dev
```
3.build a extension
```shell
npm run build
```
